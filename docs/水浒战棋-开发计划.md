## 二、M1 战斗原型（2026.08 — 2026.10，约10周）

**目标**：跑通核心战棋规则，用 3 名风格迥异的武将（林冲突进、鲁智深拉拽、安道全治疗）验证「效果原子化」框架。原型期画面一律使用占位素材，不投入正式美术。

**验收即策划文档原型期 5 个阶段，排期如下（第3阶段风险最高，预留1周缓冲）：**

### 2.1 程序排期

| 周 | 内容（对应策划文档原型阶段） | 关键产出 |
|---|---|---|
| W1—2 | 阶段1：格子棋盘 ＋ 寻路 ＋ 移动/普攻 | 工程初始化；`Grid`＋`GridCell`；AStarGrid2D 寻路（地形消耗入权重）；TileMapLayer 地形＋高亮双层；移动与普攻指令 |
| W3—4 | 阶段2：行动条 ＋ 伤害结算 ＋ Buff系统 | CTB 速度排队（AV＝1000÷速度）；行动预览条（后5—8个单位）；伤害公式（攻防、暴击、背刺/侧击、高低差）；Buff/Debuff 统一管理与每回合 tick |
| W5—7 | 阶段3：效果原子化框架 ＋ 3个示范技能 | `EffectSystem` 原子效果执行器；`SkillData` ＝效果序列＋范围模板＋目标规则；落地 `ult_fengxue`（林冲）、`ult_chuiyangliu`（鲁智深）、安道全治疗技 |
| W8—9 | 阶段4：评分AI ＋ 自动/半自动切换 | 效用评分制 `BattleAI`（基础因子×职业权重，权重表配置化）；玩家点击与AI决策走同一 `Command` 指令管道；手动/自动切换 |
| W10 | 阶段5：布阵 ＋ 胜利条件 ＋ 触发器 ＋ 整合 | 布阵阶段（部署区、拖拽上阵）；歼灭/坚守/回合耗尽判定；`LevelConfig` 资源与事件触发器（回合数/单位死亡/进入区域/血量阈值） |

**贯穿性程序任务**（W1 启动，随阶段推进补全）：

- `DataLoader`（AutoLoad）：解析三张 CSV → 生成并缓存 `UnitData` / `TerrainData` / `SkillData`，全项目按 ID 查询（数据表说明第四节）。
- CSV 数据校验工具（编辑器脚本）：校验技能引用、羁绊 ID、枚举合法性、ID 重复（数据表说明第五节）。
- `BattleManager` 状态机（DEPLOY / IDLE / MOVE_PREVIEW / TARGETING / EXECUTING / AI_TURN / BATTLE_END），状态切换走信号广播。
- 逻辑层单元测试：伤害公式、CTB 排序、原子效果执行器（测试框架选型 W1 内确定）。

### 2.2 策划任务

- 落地林冲、鲁智深、安道全 3 将的完整数据（武将表已有基础值，补齐战斗所需全字段）。
- 搭建 1 个 8×8 调试关卡（含平原/森林/山地/拒马/营帐，覆盖全部地形行为验证）。
- 按「蓝将基准、紫+8%、橙+15%、绿−8%」基调做第一轮数值 sanity check。
- 编写 AI 权重表初版（策划文档 8.2—8.4 的数值录入配置表）。

### 2.3 美术任务

- 原型只出占位素材：纯色块地形、剪影小人、临时 UI 框。
- 同步验证正式管线：以林冲基准图为 `reference_image` 产出鲁智深、安道全立绘，跑通「AI生成 → 统一后期 → 64px 验收」全流程（美术指导第六节），为 M2 批量生产定标。

### 2.4 M1 验收标准（DoD）

- [ ] 在调试关卡完成一场完整手动战斗：布阵 → CTB 行动 → 移动/普攻/技能/绝技 → 胜负结算。
- [ ] 3 个示范技能全部由 CSV 的 `effects` 字段配置驱动，程序零硬编码；新增一个技能只需加表。
- [ ] 自动战斗可整局托管，手动/自动可在对局中切换（同一指令管道）。
- [ ] 逻辑瞬时结算、表现排队播放：自动战斗 2 倍速/跳过不触碰逻辑层。
- [ ] 全部数值从 CSV 读取，策划改表重进游戏即生效。
- [ ] 伤害公式、CTB、效果执行器有单元测试覆盖。

## 三、M2 垂直切片（2026.11 — 2027.04，约24周）

**目标**：交付「第一章完整内容 ＋ 养成闭环」＋ 示范关卡「智取生辰纲」，品质达到可对外展示水准，用于找发行/众筹。

### 3.1 内容与系统分解

| 模块 | 主要任务 | 负责 |
|---|---|---|
| 养成闭环 | 武将升级/升星/突破；武器强化/精炼/宝石镶嵌；技能升级；聚义升星（品质进阶）；羁绊系统 | 程序＋策划 |
| 第一章战役 | 第一章 6—8 关（含教学序列）、剧情演出框架、过场包装（水墨仅用于UI/过场） | 策划＋程序＋美术 |
| 示范关卡 | `ch03_01` 智取生辰纲完整实现：10×8 黄泥冈地图、夺担引导机制、酒摊互动、杨志 BOSS（半血狂暴）、T1—T6 触发器、双互斥成就、晁盖/刘唐 NPC 友军 | 程序＋策划 |
| 布阵完整版 | 敌方可见阵容、危险范围热力图、最多8人上阵 | 程序 |
| 武将实装 | 首批约 10 名：新手标准队（鲁智深/武松/花荣/安道全/吴用）＋ 林冲、李逵、白胜 ＋ 生辰纲关卡所需 NPC/敌方（杨志、老都管、厢军×2） | 策划＋美术 |
| 山寨经营雏形 | 聚义厅/铁匠铺/演武场的基础产出循环（派驻机制可简化） | 程序＋策划 |
| 外围系统 | 存档系统、主菜单/章节选择、设置（音量/分辨率/键位） | 程序 |
| UI 包装层 | 宣纸底＋朱砂强调＋水墨边框的正式 UI Kit；稀有度色彩体系；伤害跳字/掉落光效同色体系 | 美术 |
| 手柄适配 | 战斗与菜单全手柄流（方形格子天然友好），为 Steam Deck 打底 | 程序 |
| 音频 | 外包启动：主题曲 1 首、战斗/营地 BGM 各 1、全套 SFX（攻击/技能/UI） | 策划对接 |
| 宣发物料 | Steam 商店页素材、60—90 秒 trailer、可下载 demo（垂直切片本体） | 全员 |

### 3.2 美术批量生产（沿用美术指导第六节管线）

- 首批 10 将：立绘（正面3/4）＋ 战斗 Q 版形象，逐将建立 prompt 卡片（固定配色词），按职业分批生成。
- 第一章场景：黄泥冈全套地形 tile（平原/松林/山地/土路/酒摊/营帐/拒马）＋ 场景装饰。
- 技能特效与演出：绝技全场动画（可跳过）至少覆盖首批 10 将。
- 每批素材过「64px 剪影验收清单」，并留存生成记录（prompt/日期/工具）备 Steam AI 披露。

### 3.3 阶段排期（24周）

| 时段 | 重点 |
|---|---|
| W1—6 | 养成系统三件套（武将/武器/技能）＋ 存档 ＋ 羁绊系统 |
| W7—12 | 第一章关卡与教学 ＋ 布阵完整版 ＋ 首批武将数据与美术量产启动 |
| W13—18 | 智取生辰纲示范关卡 ＋ 山寨经营雏形 ＋ UI 包装层全面替换占位 |
| W19—22 | 音频接入、手柄适配、性能与打磨、数值第一轮调优 |
| W23—24 | Demo 封装、商店页与 trailer、内部验收与发行/众筹投递 |

### 3.4 M2 验收标准（DoD）

- [ ] 第一章从教学到通关可完整游玩，存档/读档正常。
- [ ] 养成闭环成立：战斗 → 资源 → 升级武将/武器/技能 → 挑战更难关卡。
- [ ] 智取生辰纲关卡三条路线均可达成（药酒智取/硬打/白胜阵亡），两个互斥成就均可解锁。
- [ ] 至少 10 名武将以正式美术实装，羁绊加成生效。
- [ ] 自动战斗在全部切片关卡表现可接受（AI 不犯低级错误，权重可调）。
- [ ] 全程手柄可玩；Steam Deck 分辨率先行适配。
- [ ] Demo、商店页文案与素材、trailer 齐备，达到投递发行/众筹状态。

## 六、横向工作流与工程红线

以下规范贯穿所有阶段，任何里程碑不得突破：

1. **逻辑与表现分离**：逻辑瞬时结算、表现排队播放；自动加速/跳过只能是表现层问题。
2. **指令管道统一**：玩家输入与 AI 决策生成同一种 `Command`，进同一队列执行。
3. **信号驱动**：`turn_started` / `unit_damaged` / `unit_died` / `skill_executed` 等全部走信号，成就、教学、音效只订阅不侵入。
4. **数据全 Resource 化**：武将/技能/地形/关卡均为 `.tres`；CSV 是唯一数据源，禁止代码硬编码数值；调参＝改 CSV 重进游戏。
5. **技能零程序介入**：新技能 ＝ 技能表加一行拼原子效果；新原子效果才动 `EffectSystem`，并同步更新词表文档。
6. **美术合规**：AI 素材按固定六段式 prompt 生产，批次记录留档，验收走 64px 剪影清单。
7. **版本管理**：Git 管理代码与数据表；CSV 改动随提交说明数值意图；里程碑打 tag。

## 七、风险登记与开发对策

| 风险（策划文档第十三章） | 开发侧对策 | 监控点 |

| 「唯橙将论」 | 机制保值清单逐个验收；M3 验收含平民绞肉机队强度实测 | M3 验收 |
| AI 体验不佳 | 权重表全程配置化；半自动绝技释放条件表化（策划文档 8.5）；垂直切片期安排外部试玩收集 AI 观感反馈 | M2 试玩反馈 |
| 人手不足 / 外包延期 | 音频外包 M2 中段即启动并锁交付节点；玩法模式优先级：主线 > 日常副本 > 演武场 > 远征，延期时从尾部砍 | 双周例会进度 |
| 同质化竞争 | 剧情演出、羁绊彩蛋、绝技设计围绕市井江湖气；每章至少 1 个剧情彩蛋关卡（生辰纲为样板） | 章节验收 |
