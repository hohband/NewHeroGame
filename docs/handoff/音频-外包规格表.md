# 音频外包规格表（SFX & BGM）

> 用途：解除阻塞项 #2（音频外包）。直接作为外包需求单使用。
> 风格总基调：明快、市井江湖气（美术指导第一节）；忌暗黑、忌重交响。
> 交付格式：WAV 48kHz/24bit 母带 + OGG Vorbis q6 游戏用（Godot 直接可读）。
> 接入点已就绪：战斗内事件信号（turn_started / skill_executed / unit_died / battle_ended / dialogue），
> 收到素材后由程序按本表映射接入，无需外包方理解工程。

---

## 一、BGM（3 首，循环无缝）

| 编号 | 名称 | 时长 | 场景 | 风格要求 |
|---|---|---|---|---|
| bgm_main | 《聚义》主题曲 | 90–120s | 主界面/山寨 | 唢呐+琵琶主奏，豪迈中带市井诙谐，记忆点要强 |
| bgm_battle | 《替天行道》战斗 | 90–120s 循环 | 常规战斗 | 中速，鼓点稳，紧张但不压迫；可随战况加一层（可选分轨） |
| bgm_camp | 《大碗酒》营地/结算 | 60–90s | 结算/日常 | 轻松、快板，三弦/板胡，酒肆感 |

## 二、SFX（按游戏事件映射）

### 战斗基础
| 编号 | 事件 | 数量 | 要求 |
|---|---|---|---|
| sfx_move | 单位移动脚步 | 2–3 变体 | 短促、木质地面感 |
| sfx_atk_melee | 近战普攻 | 3 变体 | 朴刀/枪/斧三类可共用，干净劈砍 |
| sfx_atk_ranged | 远程普攻 | 2 变体 | 弓弦+破空 |
| sfx_hit | 受击跳字 | 2 变体 | 短、脆，不血腥 |
| sfx_dodge | 闪避 | 1 | 轻"嗖" |
| sfx_block | 格挡 | 1 | 金属闷击 |
| sfx_crit | 暴击 | 1 | 加重+亮色尾音 |
| sfx_die | 单位阵亡 | 1 | 倒地+鼓点收束，不惨烈 |
| sfx_heal | 治疗 | 1 | 温润上行 |
| sfx_buff / sfx_debuff | 增益/减益 | 各 1 | 一亮一暗，对比清晰 |

### UI/系统
| 编号 | 事件 | 要求 |
|---|---|---|
| sfx_ui_click / sfx_ui_hover | 按钮点击/悬停 | 木质或纸感，轻 |
| sfx_turn | 新单位行动提示 | 清脆提示音 |
| sfx_win / sfx_lose | 胜利/失败 | 胜利唢呐短句；失败低沉两拍，不沮丧 |
| sfx_levelup | 武将升级 | 上行短句 |
| sfx_unlock | 新武将加入 | 印章"钤"一声+鼓点 |
| sfx_collect | 夺取/收集（生辰纲） | 包袱落地+轻快尾音 |

## 三、验收标准

- 全部素材响度统一（-16 LUFS 游戏内目标，母带 -14 LUFS）；
- SFX 单条 ≤ 2s（绝技除外），无爆音、无直流偏移；
- 每类给 2–3 个变体的须可随机轮换不违和；
- 交付清单按本表编号命名（如 sfx_atk_melee_01.ogg）。
