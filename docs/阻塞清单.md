## 二、附录 A：桌面包导出命令

```bash
# macOS（通用二进制 dmg）
mkdir -p build/macos && godot --headless --path . --export-release "macOS" build/macos/shuihu.dmg

# Windows（x86_64 exe）
mkdir -p build/windows && godot --headless --path . --export-release "Windows Desktop" build/windows/shuihu_x64.exe
```

- 导出模板已装在 `~/Library/Application Support/Godot/export_templates/4.7.1.stable/`（本机一次性事项）。
- 产物约 120MB（dmg）/ 110MB+53MB（exe+pck）；未签名/未公证，分发时 macOS 用户需右键打开（Gatekeeper 提示属正常）。
