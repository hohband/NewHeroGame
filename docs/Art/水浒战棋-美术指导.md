# 《水浒战棋》美术指导（Art Direction）

> 版本 V1.0 ｜ 2026年7月 ｜ 配套文件：风格基准图（角色/战场）
> 本文档用途：①统一团队（及AI工具）的美术产出标准；②作为AI生成素材时的固定风格规范。

---

## 一、风格定位

| 项目 | 规范 |
|---|---|
| 总体风格 | Q版卡通渲染（2.5头身），明快饱和，高辨识度剪影 |
| 直接参考 | 《部落冲突》的角色辨识度与色彩明快感 |
| 文化调味 | 水墨元素仅用于UI、过场、包装层（宣纸底、朱砂印、墨笔触），不进入战斗单位本体 |
| 视角 | 战场：45°俯视等距视角；角色：正面3/4立绘 |
| 绝对避免 | 写实血腥、暗黑风、日漫画风、和风元素混入 |

**一句话原则**：三岁小孩能认出"这是梁山好汉"，策略玩家能一眼读懂战场。

## 二、角色设计规范

### 2.1 比例与体量

- 标准Q版 **2.5头身**；先锋/体型壮硕者可放宽至2.2头身显魁梧（鲁智深、石勇），敏捷系收紧至2.8头身显瘦高（时迁、王定六）
- 头部占视觉重心，五官简化但眉眼必须有戏（浓眉大眼为基本款）
- 描边：外轮廓粗描边，内部细节细线，保证缩小到格子尺寸（64px）仍可辨

### 2.2 武将识别度三原则

1. **绰号视觉化**：绰号必须变成可穿戴的视觉符号——豹子头林冲→豹纹头巾；黑旋风李逵→比人还宽的双板斧；一丈青扈三娘→红棉套索长飘带
2. **武器实名制**：名兵器造型固定且考据——水磨禅杖、雪花镔铁戒刀、丈八蛇矛、青龙偃月刀，全游戏只此一件，不得撞款
3. **一个夸张特征**：每名武将只允许一个夸张点（比例失衡的武器、过大的酒葫芦、拖地长袍），多则乱

### 2.3 职业视觉编码

| 职业 | 剪影特征 | 标志元素 | 主色倾向 |
|---|---|---|---|
| 先锋 | 宽肩重甲，体量×1.1 | 大盾、重甲片 | 玄铁深灰＋墨 |
| 步军 | 轻装利落，贴身短打 | 朴刀、双持、戒刀 | 赭石、粗布色 |
| 马军 | 坐骑为视觉中心 | 长枪、长矛、马具 | 甲青＋朱砂 |
| 神射 | 瘦高、披风或斗篷 | 长弓、箭囊 | 竹青＋皮棕 |
| 谋士 | 长袍大袖 | 羽扇、拂尘、葫芦 | 靛蓝道袍 |
| 医者 | 圆和体态 | 药箱、药葫芦 | 月白＋药绿 |
| 辅助 | 非战斗感 | 大旗、铁匠锤、工具 | 杂色点缀 |

### 2.4 名将与无名之辈的差异处理

- 橙/紫将：服饰细节多一层（甲片纹理、飘带、披风），配色更纯更正
- 蓝/绿将：布衣为主、细节少，但**机制道具必须醒目**（郁保四的旗、白胜的酒桶、汤隆的钩镰枪）——玩家看道具知功能

## 三、战场场景规范

- 格子边界必须清晰可读（描边或色差），任何装饰不得干扰格子识别
- 地形色块分明：草地为黄绿主调、森林深绿成团、山地灰褐岩石、水面高饱和蓝、土路暖黄——地形即颜色，颜色即信息
- 场景物件（营帐、酒摊、拒马、旗帜）尺寸不得超出一个格子，避免误读为单位
- 敌我标识：我方蓝色旗号/标记，敌方红色旗号/标记，全局统一
- 光照：统一左上光源，软阴影，禁止戏剧性打光

## 四、UI与包装层

- 底色：宣纸米色；边框：水墨笔触或简化的回纹
- 强调色：朱砂（按钮、稀有度标识、成就印章）
- 字体气质：标题用厚重书法感的黑体/楷体，正文用清晰屏显字体
- 稀有度色彩：橙将=赤金、紫将=绛紫、蓝将=靛青、绿将=竹绿（与伤害数字、掉落光效共用同一套）

## 六、AI素材生产规范

### 6.1 Prompt模板结构（固定六段式）

```
[风格锚点] + [主体描述] + [服饰/配色细节] + [视角/构图] + [渲染词] + [背景]
```

**风格锚点（固定不变，逐字复用）**：

```
chibi game character, 2.5-head-tall cute proportions, clean bold outlines, soft cel shading, vibrant Clash-of-Clans-inspired cartoon style, high quality game asset
```

### 6.2 一致性技巧

1. 以已确认的基准图为 `reference_image` 进行图生图，所有新武将都从基准图"繁衍"
2. 配色词固定（如林冲永远是 "dark teal armor with vermillion red sash"），写入每将的prompt卡片
3. 同批次生成同一类素材（一次生完全部先锋），减少模型状态漂移
4. 后期统一处理：统一描边粗细、统一调色、统一缩放至规格尺寸

### 6.3 通用负面词

```
realistic, photorealistic, dark, bloody, anime style, japanese armor, extra limbs, blurry, watermark, text, signature, oversaturated
```

### 6.4 素材验收清单

- 缩至64px剪影仍可辨认职业与武将
- 与基准图并排放置不违和
- 格子场景素材不干扰格子边界识别
- 可抠图（角色）、可切片（场景/特效）

## 七、风格基准图Prompt

### 7.1 角色基准（已生成：林冲）

```
Chibi game character concept art, 2.5-head-tall cute proportions, Song dynasty Chinese warrior from Water Margin: Lin Chong the Panther Head - leopard-print headband, determined eyes, dark teal cloth armor with vermillion red sash, holding a long spear diagonally, full body front view, standing pose, clean bold outlines, soft cel shading, vibrant Clash-of-Clans-inspired cartoon style, plain warm light beige background, character reference sheet, high quality game asset
```

**生成其他武将时，只替换中段主体描述**，例（鲁智深）：

```
...Water Margin: Lu Zhishen the Flowery Monk - bald head, large prayer beads, bare arms with floral tattoos, monk robe in dark grey-blue, holding a massive iron monk spade...
```

## 八、合规提醒

- 按Steam 2026年政策：AI辅助**代码**无需披露；但AI生成的**美术、音频、文案**内容进入游戏，必须在商店页的AI披露栏声明
- 保留每批素材的生成记录（prompt、日期、工具），以备审核与版权自证
